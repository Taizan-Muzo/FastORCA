# FastORCA qcMol Substitute Docs

This folder is the productized documentation set for the qcMol substitute mainline.

## Start Here

1. `qcmol_substitute_default.md`  
Single blessed profile and entrypoint.

2. `qcmol_canonical_surface.md`  
Frozen canonical surface with red/yellow/green dependency guidance.

3. `qcmol_consumer_guide.md`  
Minimal / enhanced / caution consumption tiers and export helper.

4. `qcmol_operational_checklist.md`  
Operator runbook and pass/fail health gates.

5. `qcmol_substitute_readiness.md`  
Readiness judgement and explicit project boundary notes.

6. `qcmol_alignment_master_table.md`  
Alignment source-of-truth and reporting commands.

## Boundary Statement

FastORCA targets an **open-source qcMol substitute**, not an exact closed-stack replica.  
Exact-only families (NAO/LI/ADCH/NPA_exact/NBO_BD/LBO) are intentionally archived as roadmap-only.
